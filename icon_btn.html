Icon Button

Square buttons for icon-only actions.

Base

Default icon button with icon-btn.

Mute Variant

Subtle color using icon-btn:mute.

Sizes

Append -sm to any variant for smaller size.

States

Disabled state.

API Reference

Selectors

Class Description
icon-btn Base icon button
icon-btn:mute Muted color variant

Modifiers

Modifier Description
-sm Small size (append to variant)

States

Attribute Description
disabled Disabled state

Usage

<!-- Base -->
<button class="icon-btn" aria-label="Menu">...</button>

<!-- Muted variant -->
<button class="icon-btn:mute" aria-label="Settings">...</button>

<!-- Muted + Small -->
<button class="icon-btn:mute-sm" aria-label="Close">...</button>

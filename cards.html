Cards

Cards

Container with padding, border, shadow. Use .pri, .sec, or .gla for variants.

Default

Subtle primary tint background.

Primary

Stronger primary color tint.

Secondary

Uses secondary color tint.

Glass

Frosted glass with backdrop blur.

Alert Cards

Alert cards with right-aligned action buttons (Apple design pattern). Defaults to info color.

Information

This is an informational alert.

Confirm Action

Are you sure you want to proceed?

API Reference

Classes

Class Description
.card Base card container
.card:alert Alert variant with info color and right-aligned action buttons

Nested Elements

Element Behavior
h3 Styled as title (no top margin, small bottom margin)
p No block margin for tighter spacing
.btn Auto-aligns to right via margin-inline-start (alert only)

Usage

<!-- Basic card -->
<div class="card">
  <h3>Title</h3>
  <p>Content</p>
</div>

<!-- Alert with single action -->
<div class="card:alert">
  <h3>Alert Title</h3>
  <p>Alert message</p>
<button class="btn pri out sm">OK</button>
</div>

<!-- Alert with multiple actions -->
<div class="card:alert">
  <h3>Alert Title</h3>
  <p>Alert message</p>
  <div class="flex:cluster gap:xs">
<button class="btn sec out sm">Cancel</button>
    <button class="btn pri out sm">Confirm</button>
  </div>
</div>

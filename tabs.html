Tabs

Pill-shaped tab container with active state.

Base

Default tabs with primary color.

Variants

Color variants using tabs:variant pattern.

Sizes

Append -sm for smaller tabs.

Full Width

Append -full to stretch tabs.

API Reference

Container

Class Description
tabs:pri Primary color
tabs:sec Secondary color

Tab Items

Class Description
tab Base tab (inactive)
tab:active Active tab state

Modifiers

Modifier Description
-sm Small size
-full Full width (tabs stretch)

Usage

<!-- Primary tabs -->
<div class="tabs:pri">
  <button class="tab:active">Tab1</button>
  <button class="tab">Tab2</button>
  <button class="tab">Tab3</button>
</div>

<!-- Small secondary tabs -->
<div class="tabs:sec-sm">
  <button class="tab:active">Tab1</button>
  <button class="tab">Tab2</button>
</div>

<!-- Full width -->
<div class="tabs:pri full">
  <button class="tab:active">Tab1</button>
  <button class="tab">Tab2</button>
  <button class="tab">Tab3</button>
</div>

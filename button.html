Button

Base

Default btn with neutral surface.

Variants

Color variants using btn:variant pattern.

Outline Modifier

Add out class for outline style. Works with any variant.

Sizes

Append -sm or -lg to variant.

States

Disabled and loading states.

With Icons

Buttons have built-in flex gap for icon + text.

API Reference

Selectors

Class Description
btn Base button, neutral surface
btn:pri Primary color
btn:sec Secondary color
btn:wrn Warning color
btn:dgr Danger color

Modifiers

Modifier Description
-sm Small size (append to variant)
-lg Large size (append to variant)
out Outline style (separate class)

States

Attribute Description
disabled Disabled state
aria-busy="true" Loading state

Usage

<!-- Base -->
<button class="btn">Default</button>

<!-- Variant -->
<button class="btn:pri">Primary</button>

<!-- Variant + Size -->
<button class="btn:pri-sm">Small Primary</button>

<!-- Variant + Outline -->
<button class="btn:dgr out">Danger Outline</button>

<!-- Variant + Size + Outline -->
<button class="btn:sec-lg out">Large Secondary Outline</button>

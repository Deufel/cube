Popup

Anchor-positioned popover with animations. Uses CSS Anchor Positioning + Popover API.

Base Menu

Click to open an anchored popup menu.

Positioning

Anchor positioning is set via inline styles. Common patterns:

API Reference

Selectors

Class Description
popup Base popup container
popup:menu Menu variant (grid layout)
popup-item Menu item with hover state

Required Attributes

Element Attribute
Trigger popovertarget="id", style="anchor-name: --name"
Popup popover, style="position-anchor: --name"

Usage

<!-- Trigger -->
<button popovertarget="my-menu" style="anchor-name: --my-btn">
  Open Menu
</button>

<!-- Popup -->
<div id="my-menu" class="popup:menu" popover
     style="position-anchor: --my-btn; top: anchor(bottom); left: anchor(left);">
  <button class="popup-item">Home</button>
  <button class="popup-item">Settings</button>
  <button class="popup-item">Logout</button>
</div>

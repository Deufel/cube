Badge

Notification indicator for icons and buttons.

Base

Default badge with info color. Position relative to parent.

3
12

Variants

Color variants using badge:variant pattern.

3
3
3
3

Dot Exception

Append -dot for a minimal dot indicator.

API Reference

Selectors

Class Description
badge Base badge, info color
badge:suc Success color
badge:wrn Warning color
badge:dgr Danger color

Exceptions

Modifier Description
-dot Dot indicator (no content)

Usage

<!-- Parent must have position: relative -->
<div style="position: relative;">
  <button class="icon-btn">...</button>
  <span class="badge">3</span>
</div>

<!-- Color variant -->
<span class="badge:dgr">5</span>

<!-- Dot indicator -->
<span class="badge:suc-dot"></span>

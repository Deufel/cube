Chips

Non-interactive labels for categorization and status display.

Base

Default chip with primary tint.

Default

Variants

Status variants using chip:variant pattern.

Info Success Warning Danger

Removable

Add .close span for dismissible chips.

Default × Info × Success × Warning × Danger ×

API Reference

Selectors

Class Description
chip Base chip, primary tint
chip:inf Info (blue)
chip:suc Success (green)
chip:wrn Warning (yellow)
chip:dgr Danger (red)

Modifiers

Modifier Description
-sm Small size (append to variant)

Inner Elements

Class Description
.close Dismissible close button

Usage

<!-- Base -->
<span class="chip">Default</span>

<!-- Variant -->
<span class="chip:inf">Info</span>

<!-- Variant + Size -->
<span class="chip:suc-sm">Small Success</span>

<!-- Removable -->
<span class="chip:dgr">Error <span class="close">×</span></span>
